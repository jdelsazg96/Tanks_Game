Tanks es un juego implementado en Python, se recomienda tener descargada la aplicación PyCharm junto con el intérprete Anaconda para una correcta ejecución del programa.

Además también se debe tener instaladas las librerías de Pygame, necesarias para poder compilar correctamente el programa.

Finalmente, para que no exista ningún problema, se deben descargar todos los documentos (básicamente fotografías) para la adecuada representación gráfica del juego.
